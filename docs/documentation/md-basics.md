# Markdown Basics

Markdown is a lightweight markup language used to format plain text. Itâ€™s simple, human-readable, and can be converted to other formats like HTML or PDF.

## Key Syntax
### Headings
Use `#` to `######` for headings:
```markdown
# Heading 1
## Heading 2
### Heading 3
```

### Text Formatting
- **Bold**: `**text**`
- *Italic*: `*text*`
- ~~Strikethrough~~: `~~text~~`

### Lists
- **Unordered**: Use `-` or `*`.
- **Ordered**: Use numbers with periods (`1.`, `2.`).

### Links and Images
- **Link**: `[text](url)`
- **Image**: `![alt text](url)`

### Code
- Inline: `` `code` ``
- Block:
    ```markdown
    ```
    Code block
    ```
    ```

### Tables
```markdown
| Header 1 | Header 2 |
|----------|----------|
| Data 1   | Data 2   |
```

Learn more in the [Markdown Guide](https://markdownguide.org/).
